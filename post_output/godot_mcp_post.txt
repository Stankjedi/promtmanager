[정보] AI로 Godot 게임 엔진 자동화하는 MCP 서버 공유한다 (godot-mcp-omni)

Godot 쓰는 애들 중에 AI 어시스턴트 제대로 써보고 싶은 놈들 있을 것 같아서 가져옴.
이번에 godot-mcp-omni라고 MCP 서버가 업데이트됐는데, 기존 프로젝트 포크해서 기능 엄청 때려 박았더라.
AI가 직접 에디터 만지게 하거나 헤드리스로 작업 돌릴 때 유용해 보임.

주요 내용 요약해준다:

1. 5대 통합 관리자 시스템 (v0.2.0)
- Scene Manager: 노드 생성하고 계층 구조 수정 가능.
- Inspector Manager: 프로퍼티 값 검색하고 수정하는 거 가능.
- Asset Manager: 에셋 리소스 관리 전용.
- Workspace Manager: 파일 시스템 접근이랑 코드 검색.
- Editor View Manager: 이게 좀 꿀인데, 현재 개발 화면 PNG로 찍어서 AI한테 보내주거나 화면 전환, 스크립트 에디터 제어까지 됨.

2. 하이브리드 작업 방식
- 에디터가 켜져 있으면 RPC로 실시간 동기화해서 제어함.
- 에디터 꺼져 있으면 알아서 Headless 모드로 전환해서 명령행으로 작업 수행한다.

3. 편의 기능
- 자동 타입 캐스팅: AI가 JSON으로 대충 던져줘도 Godot의 Vector3나 Color 같은 타입으로 알아서 변환해서 박아줌.
- CI/CD 최적화: 아토믹 배치 작업 지원해서 자동 빌드 돌릴 때 편함.

요구 사양:
- Node.js 20 이상
- TypeScript 5.3 이상
- Godot 4.4 이상 (에셋 파이프라인 호환성 때문인 듯)

확인 필요:
- Godot 4.4 베타 사양 기준이라 하위 버전에서는 안정성 확인 필요함.

기존에 그냥 코드만 짜주던 AI랑 다르게 엔진 내부를 직접 건드리는 방식이라 생산성 차이 꽤 날 듯하다.
Godot으로 특이점 앞당길 놈들은 한번 찍어먹어봐라.
